##Wolfenstein-clone
Raycasting Engine
A pseudo-3D engine built in Python and Pygame that uses raycasting to transform a 2D world into a 3D perspective.

(Special thanks to: "Coder Space", youtube channel:"https://www.youtube.com/@CoderSpaceChannel")

Overview
This project is a functional "Wolfenstein 3D" clone. While the core engine is finished, I plan to update it as I gain more advanced skills and knowledge in computer graphics.

Features
Functional Raycasting Engine: Real-time calculation of wall distances.

Dynamic Image Scaling: Adjusts wall height based on player distance.

360-Degree Movement: Full rotation and fluid movement using both keyboard and mouse.

Pseudo-3D Projection: Transforming 2D grid data into a 3D visual experience.

🛠️ Tech Stack
Language: Python

Library: Pygame

The Learning Process
Coming into this project, I only knew the very basics of Python. By following the "Coder Space" guide, I focused on active learning—understanding the "why" behind every line of code rather than just copying and pasting.

This project pushed me to develop a new type of problem-solving mindset. As the development progressed, I began writing my own custom logic to expand the engine's capabilities and match the core architecture.

Skills Learned
Trigonometry & Circle Math: Calculating angles, distances, and projection planes.

Object-Oriented Programming (OOP): Understanding classes, objects, and why we use them to organize game entities.

Modular Programming: Learning how to import logic from external files to keep the main.py clean.

Game Loops: Mastering while loops to handle real-time rendering and input.

Logic Flow: Understanding how different functions interact to form a cohesive program.
